# OpenClaw 语音网关 MVP 宣传视频脚本 (V2.0 炫酷动态版)

> **核心目标**：向极客开发者及企业高管展示 ZEGO AI Agent + OpenClaw 结合后的强悍通信底座能力（低延迟、打断、极速长短记忆并发、快慢分离架构），建立未来 LUI 的标杆认知。
> **整体基调**：极客科幻感（类似钢铁侠 Jarvis 结合 macOS / 新版 Siri 光效），展现从沉浸式的高端流体交互瞬间落地到一行解决战斗的代码，形成强烈反差。
> **演练原型**：请直接使用浏览器（推荐移动设备调试模式）打开并体验 `doc/promo_prototype.html` 获取真实流式交互体感。

## 1. 视听基准设定 (基于 V4 Prototype 实现)

*   **视觉风格 (Aurora & Glassmorphism)**：
    *   **深空极光背景**：摒弃死黑，底层采用 3 个缓慢游离的高斯模糊（Blur-120）极光球。背景光晕的色温会随着当前语音的收发状态（湖青/幽蓝/警告橙）发生情绪共鸣式的渐变。
    *   **主视觉核心 (Fluid Voice Core)**：取消死板的录音机波形条。屏幕正中心是一颗**三层发光的科幻流体能量球**。待机时微弱呼吸；听到或输出时（打字机运作中）会伴随文字吐出的极速频率产生脉冲式的激震（Pulse）。
*   **UI 交互细节**：
    *   **玻璃态悬浮舱 (HUD Widget)**：右上角带有极大毛玻璃虚化（backdrop-blur-24px）和极细高亮反光边的面板。作为“硬核技术自证”，真实播报后台并发 Webhook 任务的真实进度，体现快慢分离、全双工不阻塞。
    *   **动态流式字幕弹匣 (Auto-scroll Subtitles)**：屏幕下方的对白字幕不再生硬闪现，而是采用拟真的打字机逐字输出（带有类似真实语速的毫秒级抖动和标点停顿）。最新的一个回合（User + AI）保持高亮清晰悬挂，过去的对话会慢慢向上推移，并逐渐缩小、变灰并产生重影高斯模糊，直到消失，极具纵深压迫感。

> **概念效果参考**：极简能量内核为主体，后台长耗时任务通过不抢占焦点的玻璃态悬浮卡片展示进度。这极好地体现了我们所宣扬的“快慢分离、全双工不阻塞”架构。

## 3. 技术自证与备注说明 (防伪设计)

在单页面 H5 原型（`promo_prototype.html`）的编码和设计上，我们为演示者配置了极强的“自解说防御力”：
1. **真实语速映射**：底层的流式打字算法（Typing Engine）并没有使用匀速输出，而是模拟了 ASR 的微卡顿与 TTS 的断句。且字体的吐出严格与中间主视觉核心球的“抽搐式脉冲放大”绑定，呈现极为细腻的**“音画同步与声浪共鸣”**。
2. **状态透明化栈 (Widget Log)**：右上角高糊的毛玻璃挂件实时显示着后端的任务状态从 `RUNNING` 变为 `SUCCESS`。以此向硬核观众自证：前台 AI 在回话和倾听时，后台进度条是完全独立在走动的，证明了代码底层的异步并发队列设计。
3. **防演砸保护（Double-Tap Skip）**：H5 逻辑通过劫持屏幕全局 Tap，支持如果在某一句长文流式加载中再次点击，能够“0延迟瞬间输出全部补齐”以跳过过场动画，确保在真实的商务路演或内部分享时，演讲者能绝对控制时序节奏，不被特效拖慢步调。
