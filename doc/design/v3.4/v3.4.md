# Fast Agent V3.4 演进设计方案

## 1. 场景化 Skill 动态拼装
支持动态加载与拼装不同的 skill 策略以快速适配多场景的交互调整：
 - **哄睡模式**: 语速放缓、高情感价值 feedback、主动引入白噪声或 ASMR 辅助。
 - **讲故事模式**: 结构化剧情叙事、抑扬顿挫的 TTS 停顿控制、情节分段的交互响应。

## 2. 动态热词修复 (零阻塞 ASR 纠错)

### 2.1 痛点与背景
在日常语音互动中常常遭遇**同音词幻觉死循环**。例如，用户企图纠正：“不是武松，是雾凇”，ASR却会因读音相同死板地输出：“不是武松，是武松”。传统方案很容易在此抓瞎，而一旦为了纠正开启阻塞式的任务模式去查错，又会导致极为严重的对话卡顿和冷场。

### 2.2 架构解法：前后台并联，无缝修复
在坚守 V3 原子脱耦和极速返回的底线要求下，该特性采用“**前台高情商硬接，后台异步搞基建**”的设计：

1. **瞬时前台秒回 (SLC)**:
   - 继续放行 `Chat Mode`。不阻塞任何意图流。
   - 依托轻量级模型的庞大常识网，当看到“武松”和“冬天水挂”同时出现时，自动推导并以最优文案自然接茬（“哦哦你说的是雾凇呀”），保证当前帧的交流毫秒级畅通。
2. **后台暗中侦破 (SLE)**:
   - 在每次交互背后的隐性异步 `SLEEngine` 钩子中，植入独家提示词教习 LLM 识别“同音幻觉”（即 `“错词” + “上下文线索” -> 推理出“真词”`）。
   - 让 SLE 静默调用本地扩展工具 `correct_asr_hotword(wrong, correct)`。
3. **平台级收音纠偏 (ToolResultHandler + CallManager)**:
   - 提取后台捕捉到的目标词（如 `雾凇`），加入房间上下文 `CallState.dynamicHotwords`。
   - 利用 `ZegoApiClient.updateAgent` 从底层更新 ASR 语料系统 Params（如：提供阶梯权重），为用户的下一句话清零误别率。
4. **防污染记忆洗稿 (DialogueMemory)**:
   - 在 `aliasMap` 内部维护局部词典`<武松, 雾凇>`。向生成阶段喂历史会话流时，执行全自动拦截并更正，解决残存错乱信息的污染。

## 3. 支持输出情感标签
- 利用现有机理增加基于语义和情境演算的低延迟情绪特征（Tags），反馈给前台前端渲染节点，深度驱动拟真 UI（如：Canvas 呼吸态、高精度虚拟人微表情等）。

## 4. 最新进展与实装详情 (V3.4.4)

已于 2026-03-21 完成全量代码实装与逻辑验证，核心特性如下：

### 4.1 提示词中心化 (Centralized Prompts)
- **位置**: `src/agent/prompts.ts`
- **逻辑**: 抽象出 `ASR_CORRECTION_DIRECTIVE_TEMPLATE` 与 `buildShadowThought`。
- **效果**: 逻辑代码 (SLC/SLE) 仅负责数据流转，文案与潜意识引导由 `prompts.ts` 统一输出。实现了“给情报，不给台词”的灵魂驱动理念，让表达权回归 LLM。

### 4.2 阶梯式动态提权 (Adaptive Weighting)
- **机理**: 为防止过度纠偏导致 ASR 侧偏，实施 **5 -> 8 -> 11** 的阶梯提权策略。
- **算法**: 
  - 首次识别错误：分配权重 **5** (初步识别引导)。
  - 连续二次错误：分配权重 **8** (加强权重，对抗干扰)。
  - 三次及以上持久性幻觉：分配权重 **11** (ZEGO ASR 侧最高优先级强制纠偏)。

### 4.3 播报冗余拦截与潜意识缝合 (Cognitive Filtering)
- **智能拦截**: 在 `ToolResultHandler` 中增加 `alreadyCorrected` 标记位。同一词对在同一次通话中重复触发时，仅执行后台权重同步，不再触发 Canvas 摘要更新，避免冗余播报。
- **人话告知**: 首次纠正时，通过 `Shadow Thought` (潜意识) 注入指令。
  - *内部指令示例*：`[ASR 纠错情报]：刚才你把“武松”误听成了“雾凇”，请用你 natural 的身份顺便告诉用户你刚才听错了。`
  - *实际表现*：AI 会根据当前人设（Persona）自主组织对白（如：“哎呀，刚才我耳朵出小差了...”），彻底消除机器人播报感。

### 4.4 记忆净化与系统自愈 (Clean Memory & Self-healing)
- **记忆洗稿**: 在 `FastAgentV3` 加载历史记录时，自动根据 `aliasMap` 进行实时文本替换，防止 ASR 错词污染后续对话生成的上下文。
- **隧道自愈**: `index.ts` 增加了 `Tunnel Watcher` 定时器，自动检测公网 URL (Pinggy) 漂移并实时同步至 ZEGO 服务端，确保护调隧道始终可用。
- **持久化**: 纠错映射表会随通话挂断自动落盘至 `voice_plugin_aliases.json`。
